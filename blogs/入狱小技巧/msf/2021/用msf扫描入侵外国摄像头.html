用msf扫描入侵外国摄像头

摸鱼人 2021-01-30 msf

# 打开kali，启动msf

  • 打开命令行输入msfconsole，即启动msf

  • 输入命令use auxiliary/gather/shodan_search

# 登陆到shodan获取apikey

  • 点击进行注册并登陆，随后点击右上角的My Account

  • 复制红线处的API KEY

  • 命令行输入set SHODAN_APIKEY 加上自己账户的apikey run

  • 再输入命令：set QUERY “webcamxp”

  • 最后输入run等待查询结果

  • 看到这个界面就证明成功了

  • 此时复制类似于红线处的一个个链接

  • 打开浏览器，粘贴进地址回车，即可看到摄像头画面了

  • 这些所有链接都是我们搜索到的大量网上泄露出来的摄像头

# 仅供学习参考，禁止非法使用
